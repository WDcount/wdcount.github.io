1.

  • OC 中的方法和 C 语言中的函数一样，都是用来保存一段特定功能的代码。
  • C 语言中定义一个函数，分为声明和实现。声明写在.h中，实现写在.c 中。
  • OC 中定义一个方法，也分为声明和实现，声明写在@interface 中，实现写在@implementation 中。
  • 编写 C 语言函数 / OC 方法的规律：
    1. 确定函数名称。
    2. 确定形参。
    3. 确定返回值。
    4. 确定返回值类型。
  • C 语言的函数分为两种：内部函数和外部函数。
  • OC 中的方法也分为两种：类方法和对象方法。
  • 类方法只能类名调用，对象方法只能用对象调用。
  • OC 中的类方法用 + 表示，对象方法用 - 表示。

  • - (void)about;

    • OC中的方法，如果没有形参不需要写()，而是直接一个;

    • 为什么没有形参不需要写()呢？因为 OC 方法中的()是用来括住数据类型的。

  • NSLog(@"型号 = %f, cpu = %i, 尺寸 = %f, 颜色 = %i", _model, _cpu, _size, _color);

    • 如果在对象方法中想访问该对象的属性，可以直接_属性名即可。
